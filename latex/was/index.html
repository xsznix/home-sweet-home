IB WAS LaTeX Template

Microsoft Word's formatting can be confusing sometimes. Fortunately, using LaTeX to write your paper can help you save a lot of trouble getting everything correct. The files are well-documented and they fully conform to Ms. Saenz's specifications on formatting as far as I know.

LaTeX is a powerful typesetting engine that is used widely in academia. If you plan on going into science or math, chances are you'll use LaTeX some time in your career. By separating the content from the formatting, you can focus on making the formatting once and then apply it everywhere in your content. And when you actually come to writing your content, you can forget about getting everything perfect; it's already been done.

LaTeX does not rely on and is not a replacement for Microsoft Word. It is not WYSIWYG and uses plain text files instead of a proprietary file format. Essentially, you can think of it as taking the contents of a text file, reads it, and outputs a PDF.

Advantages

  • Stop fighting Word! All of the margins, spaces, indents, headers, footers, and citation styles are set for you. Just write and don't worry about formatting.
  • Use the powerful BibTeX citation engine to automatically format your citations.
  • An automatic and accurate word count shows up in the correct spot at the top of the page. It counts every word in the essay except for footnotes. Words in the header aren't counted, either.
  • Automatically generate bibliography and bibliographical citations.

Installing LaTeX

If you are on Windows, you need to install MiKTeX. It is a simple package installer; download, hit install, and you're set.

If you are on OS X, you need to install MacTeX, which is pretty much the exact same thing. Download, install, and you're set.

If you are on Linux, install TeXLive.

Setting up a writing environment

To edit LaTeX files, you will probably want an editor. Notepad is fine, if you are into that sort of thing, but I highly recommend Texmaker, which has a nice assortment of features, including the all-important spell-check.

You will have to make one configuration setting:

  1. Go to Texmaker preferences.
  2. In the Commands tab, find the box labelled LaTeX.
  3. In the contents of that box, replace latex with xelatex.
    On my system (OS X), this makes the value of the box "/usr/texbin/xelatex" -interaction=nonstopmode %.tex.
  4. Click OK.
  5. At the top of the screen to the right of the first arrow, make sure you have selected LaTeX and not PDFLaTeX as your rendering engine.

This change is to use the XeLaTeX engine instead of the default PDFLaTeX. XeLaTeX supports system fonts; PDFLaTeX doesn't.

Installing the template

Download the template and extract it to any location you want; this should be the folder that you will do your WAS work in.

Using the template

The file that contains your assignment is called Template.tex. You will put your assignment/essay/whatever in this file, within the lines \begin{essaybody} and \end{essaybody}. Open this file in Texmaker to edit it.

LaTeX reads Sources.bib for bibliographic references. You may want to use a bibliography editor such as JabRef or BibDesk to manage your references. These references are accessible from all of the .tex files in the folder it is in.

Keep all of your files in the same folder. You probably want to make a copy of the Template.tex file, as you will do multiple assignments in WAS using the same template.

To learn about how to use LaTeX, use an online resource, such as the WikiBooks book on it.

Quirks

  • To cite a source, use the command \autocite{work}, where work is the name of the work you are citing as defined in the BibTeX file.
  • To cite a source at a specific pagenumber, use the command \citepage{work}{page}, where work is the name of the work you are citing and page is the page number(s).
  • You will have to run BibTeX every time you update your bibliography.
  • You may have to run LaTeX twice every time to ensure that the word count and bibliography reflect the latest changes.

If you can manage all of that, you are ready to become a LaTeX superstar! Good luck!
